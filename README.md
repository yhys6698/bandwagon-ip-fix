## 先搞清楚：你的 IP 到底有没有被封

很多人跳过这步直接去换 IP，后来发现 IP 根本没被封，问题出在别的地方。白折腾一趟。

搬瓦工 KiwiVM 后台内置了一个 IP 检测工具，这是最直接的判断方式。

**操作步骤：**

1. 登录你的搬瓦工账户，进入对应 VPS 的 KiwiVM 控制面板
2. 在 KiwiVM 里找到你的 VPS ID（一串数字，首页就有）
3. 在浏览器新标签里访问：`https://kiwivm.64clouds.com/你的VPS-ID/main-exec.php?mode=blacklistcheck`（把"你的VPS-ID"替换成实际的数字）
4. 点击页面上的 **Test Main IP** 按钮，等几秒钟出结果

结果有两种：

- 显示绿色 **IP NOT BLOCKED**：IP 没被封，连不上的原因要去别处找
- 显示红色 **IP BLOCKED**：确认被封，可以进行下一步处理

还有一种情况，结果显示"近期曾被屏蔽，预计 48 小时内解除封锁"——这表示 IP 正在慢慢恢复，可以等等看。

## 常见错误代码解读：你看到的那串英文是什么意思

搬瓦工更换 IP 或换机房时，会碰到几个高频错误提示，大部分人不知道这些代码具体意味着什么。

### 734152：Migration backend is currently not available for this VPS

这是出现最多的错误。字面上说"迁移后端当前不可用"，但实际大多数情况下原因只有一个——**你的 IP 被封了**。

IP 被封之后，系统会禁止你通过换机房的方式"绕道"换 IP。这是搬瓦工的规则：换机房功能是用来优化线路的，不是用来规避 IP 被封后果的。所以系统直接锁住了迁移入口，但给的提示不是"IP 被封"，而是这个隐晦的 734152。

另一个可能性是 KiwiVM 迁移系统临时故障。不过这个概率小得多，而且过 10-15 分钟重试一般就好了。如果连续多次、多天都出现这个报错，基本可以锁定是 IP 问题。

先去用上面那个检测工具确认 IP 状态，再决定怎么处理。

### 734151：Migration system is currently not available

系统临时故障，等几分钟重试就行。

### 3009005：KiwiVM control panel is temporarily unavailable

KiwiVM 面板维护，等待恢复。

## IP 被封的常见原因（避坑）

换了新 IP 之后，不注意使用方式，可能很快又被封。简单说几个高发场景：

- 用 VPS 跑了大量异常出站流量，触发封锁
- 分配到的 IP 本来就是之前用户留下的"历史问题 IP"，可以在拿到服务器第一时间就检测一次，有问题立即提工单
- IP 所在的地址段里其他 IP 被封了，顺带影响到你（连坐情况）
- 某些端口被专门盯住，80、443 端口被封的后果最严重

用下来感觉最稳的使用习惯就是：不跑奇怪的东西，流量模式不要太异常，尽量挑质量好的线路套餐，比如 CN2 GIA 这类线路的 IP 池质量通常比普通线路更稳。

## 换 IP 之后的几件事

换了新 IP，别忘了同步处理以下几项：

1. **更新 SSH 配置**：把 `~/.ssh/config` 或连接工具里的 Host 地址改成新 IP
2. **更新域名解析**：如果服务器绑定了域名，去 DNS 服务商把 A 记录指向新 IP，等待生效（TTL 时间内都会有缓存）
3. **重新测试检测工具**：换完 IP 后用上文的 KiwiVM 检测链接再跑一次，确认新 IP 的黑名单状态
4. **更新代理客户端配置**：如果客户端配置文件里写了 IP，逐一更新

不用重装系统，也不用重新部署服务，数据和配置都还在。

## FAQ

**Q：IP 被封后还能用 VPS 吗？**

VPS 本身正常运行。从国外访问完全没问题，从国内大陆只是连不进去。如果你有其他境外节点可以中转，照样能用；如果只能从国内直连，就需要先换 IP。

**Q：搬瓦工换 IP 多久能完成？**

付款后通常几分钟到几小时内完成，官方表示最长 24 小时内处理。实际体验来看，大多数情况下等个半小时就好了。

**Q：IP NOT BLOCKED 但还是连不上，是什么原因？**

几个可能：① SSH 端口被封（不是 IP 整体被封，只是端口被打掉）；② VPS 本身没开机或服务挂掉；③ 本地网络问题（换一个网络试试）；④ SSH 配置本身有错误。可以用 tcping 工具单独测一下 SSH 端口是否可达，区分 IP 封锁和端口封锁两种情况。

**Q：换机房和换 IP 有什么区别？**

换机房=免费，但会分配该机房对应的 IP（可能也是被封的，或者切回同一机房 IP 还是原来那个）。换机房的核心目的是换线路和换机房地点，不是专门为了换 IP 设计的。IP 被封之后，换机房功能被系统锁住，必须先付费换 IP。
